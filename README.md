## 🚀 Rules

- 협업 학습: C++과 JavaScript 동료들과 주차별 문제 풀이 진행 (총 3명)  
- 미팅 : 수요일 & 일요일  
  - 미팅 진행 시, 문제를 하나에서 두 개 정도 더 고민하고 직접 코드로 구현.  
- 코드리뷰 : Python, C++, JS 간 논리 차이를 비교하고 최적화 방법을 토론.  
- AI 사용 금지: 순수 문제 해결 능력 강화를 위해 AI 사용 금지.  
  - 단, 최소 1시간 이상 진행 후 진전이 없으면 참고 허용.
